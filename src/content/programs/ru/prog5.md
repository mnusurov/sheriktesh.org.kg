**Развитие санитарии и гигиены в сельской местности Кыргызстана**

Реализация данного проекта осуществлялась в тесном партнерстве с консалтинговой компанией WS Atkins International, через которую Британский Департамент по Международному Развитию - основной донор - предоставлял финансирование. Общественное объединение "Каунтерпарт-Шериктеш" обеспечивало консультационную и административную поддержку проекта. Так, например, методология проведения совместной оценки потребностей сообществ была предложена ОО "Каунтерпарт-Шериктеш".

**Оказанные услуги:**

-   Определение специфичных приоритетов для развития общин с помощью оценки с участием всей общины
-   Повышение потенциала для проектов по планированию инфраструктуры
-   Повышение потенциала и развитие инструментов реализации: обучение/образование
